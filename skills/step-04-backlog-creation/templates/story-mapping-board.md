## 🛡️ Guardrails & Socratic Audit
*   **No Orphans:** Verify that every Epic/Feature sits under a valid activity column.
*   **Vertical Value Slice:** Verify that each Release contains a full left-to-right walkthrough of the backbone, delivering end-to-end value.
*   **Socratic Review:** Ask the team: *"Can a user achieve their primary goal using ONLY the cards in Release 1? If not, what is missing from our Walking Skeleton?"*
